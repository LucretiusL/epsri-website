## 关于我们

东太平洋科学研究院（EPSRI）是一所致力于化学前沿基础研究与应用转化的独立科研机构，聚焦以下四大核心方向：

- 🔥 **热电材料** — Bi₂Te₃基热电材料、ZT值突破2.0
- 🧱 **金属-有机框架（MOF）** — CO₂捕获、催化转化、超高比表面积材料
- ⚙️ **先进粉体** — 纳米粉体制备、精密粒径控制、产业化应用
- 🌿 **农药天然产物合成** — 绿色农药、手性催化、天然产物全合成
